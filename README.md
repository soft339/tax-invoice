# 세금계산서 발행 관리 시스템

전자세금계산서 발행 업무를 효율적으로 관리하는 Windows 전용 프로그램입니다.

---

## 다운로드

**최신 버전 다운로드**

👉 [tax-invoice-Setup-1.0.0D.exe](https://github.com/soft339/tax-invoice/releases/latest/download/tax-invoice-Setup-1.0.0.exe)

또는 [전체 릴리즈 목록](https://github.com/soft339/tax-invoice/releases/latest)에서 다운로드

---

## 주요 기능

- **전자세금계산서 발행** — 볼타(Bolta) API 연동, 국세청 전자 발행
- **일괄 발행** — 최대 10건 동시 발행, 템플릿 저장/불러오기
- **발행 이력 조회** — 월별/사업자별 필터링
- **다중 사업자 지원** — 사업자별 인증서 관리
- **REC장기계약 청구** — REC 장기계약 세금계산서 발행 (부가기능)
- **현대건설 VPPA청구** — REC 청구서 자동 제출 및 세금계산서 발행 (부가기능)

---

## 설치 방법

1. 위 다운로드 링크에서 설치 파일을 받습니다.
2. `tax-invoice-Setup-x.x.x.exe` 실행
3. 설치 경로 선택 후 **설치** 클릭
4. 설치 완료 후 프로그램 실행 → 로그인

---

## 시스템 요구사항

- Windows 10 / 11 (64bit)
- 인터넷 연결 필요
