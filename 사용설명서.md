## 1. 시작하기

### 1.1 프로그램 설치

1. 설치 파일(`tax-invoice-Setup-x.x.x.exe`)을 다운로드합니다.
2. 설치 파일을 더블클릭하여 실행합니다.
3. 설치 경로를 선택하고 **설치** 버튼을 클릭합니다.
4. 설치 완료 후 바탕화면 아이콘 또는 시작 메뉴에서 프로그램을 실행합니다.

---

### 1.2 로그인

프로그램을 실행하면 로그인 화면이 나타납니다.

1. **이메일**과 **비밀번호**를 입력합니다.
2. **로그인** 버튼을 클릭하거나 비밀번호 입력 후 **Enter** 키를 누릅니다.

**로그인 오류 메시지 안내**

| 메시지 | 원인 |
|--------|------|
| 이메일 또는 비밀번호가 올바르지 않습니다 | 입력 정보 확인 필요 |
| 비활성화된 계정입니다 | 담당자에게 문의 |
| 사용 기간이 만료되었습니다 | 담당자에게 문의 |

---

### 1.3 화면 구성 안내

로그인 후 메인 화면은 다음과 같이 구성됩니다.

```
+------------------------------------------+
|  세금계산서 발행 관리 시스템  [테스트모드]   |  ← 타이틀바
+------------+-----------------------------+
| [대시보드]  |                             |
| [세금계산서]|                             |
| [일괄 발행] |       메인 콘텐츠            |
| [발행 이력] |                             |
| [사업자]   |                             |
| [거래처]   |                             |
| [환경 설정] |                             |
| ─────────  |                             |
| [REC장기계약] (부가기능 활성화 시)          |
| [VPPA청구]  (부가기능 활성화 시)           |
| ─────────  |                             |
| [사용설명서]|                             |
+------------+-----------------------------+
```

- **타이틀바**: 현재 모드(테스트/운영), 로그인 사용자, 비밀번호 변경 버튼 표시
- **사이드바**: 메뉴 항목 클릭으로 화면 전환
- **대시보드**: 이번달 발행 통계 및 최근 발행 내역 확인
- **부가기능 메뉴**: 환경 설정에서 사용 여부를 체크해야 표시됩니다

> **테스트 모드**에서는 실제 국세청에 전송되지 않습니다.

## 2. 세금계산서 발행

### 2.1 발행 사업자 선택

발행 화면 상단 **공급자 > 상호** 항목의 드롭다운에서 발행할 사업자를 선택합니다.
선택하면 사업자번호, 대표자명, 업태, 종목, 이메일이 자동으로 입력됩니다.

> 사업자가 목록에 없으면 **사업자 관리** 메뉴에서 먼저 등록하세요.

---

### 2.2 공급받는자 정보 입력

**방법 1 — 거래처 선택**
- **공급받는자 > 상호** 상단 드롭다운에서 등록된 거래처를 선택하면 자동 입력됩니다.

**방법 2 — 직접 입력**
- 사업자번호, 상호, 대표자명, 이메일을 직접 입력합니다.

**방법 3 — 붙여넣기 자동 입력**
- 화면 상단의 **발행 내용 붙여넣기** 영역에 메신저나 이메일로 받은 발행 안내 내용을 그대로 붙여넣고 **자동 입력 →** 버튼을 클릭합니다.
- 사업자번호, 종사업장번호, 공급가액, 세액, 이메일이 자동으로 채워집니다.

**종사업장번호**
- 종사업장이 있는 경우 3~4자리 번호를 입력합니다. (예: `175` 입력 시 `0175`로 자동 변환)

---

### 2.3 품목 입력 및 금액 계산

| 항목 | 설명 |
|------|------|
| 월 / 일 | 거래 발생 월, 일 |
| 품목명 | 공급 품목 이름 |
| 규격 | 품목 규격 (선택) |
| 수량 | 수량 입력 |
| 단가 | 단가 입력 시 공급가액 자동 계산 |
| 공급가액 | 직접 입력 가능 |
| 세액 | 공급가액의 10% 자동 계산, 수동 수정 가능 |

- **+ 품목 추가** 버튼으로 행을 추가할 수 있습니다.
- 행 우측 **×** 버튼으로 해당 품목을 삭제합니다.
- 하단에 공급가액 합계, 세액 합계, 청구금액이 자동으로 표시됩니다.

---

### 2.4 영수 / 청구 구분

| 구분 | 의미 |
|------|------|
| **청구** | 대금을 아직 받지 못한 경우 (기본값) |
| **영수** | 대금을 이미 받은 경우 |

---

### 2.5 발행 전 미리보기

**인쇄** 버튼을 클릭하면 실제 세금계산서 양식으로 미리보기 및 인쇄가 가능합니다.

---

### 2.6 발행 및 결과 확인

1. **발행하기 →** 버튼을 클릭합니다.
2. 발행 내용 확인 팝업에서 내용을 검토하고 **확인**을 클릭합니다.
3. 발행이 완료되면 **승인번호**가 표시되고, 공급받는자 이메일로 세금계산서가 자동 전송됩니다.
4. 발행 실패 시 오류 메시지를 확인하고 내용을 수정한 후 재시도합니다.

---

## 3. 일괄 발행

### 3.1 일괄 발행 화면 구성

최대 **10건**을 한 화면에서 동시에 발행할 수 있습니다.

| 열 | 설명 |
|----|------|
| 공급자 > 상호 | 발행 사업자 선택 |
| 공급받는자 > 상호 | 거래처 선택 |
| 종사업장 | 종사업장번호 (해당 시 입력) |
| 작성일자 | 발행일 |
| 발행구분 | 청구 / 영수 선택 |
| 품목명 | 품목 이름 입력 |
| 공급가액 | 금액 입력 |
| 세액 | 자동 계산 (수정 가능) |
| 발행상태 | 대기 / 전송중 / 완료 / 실패 실시간 표시 |

---

### 3.2 항목 입력 방법

1. 각 행의 공급자와 거래처를 드롭다운에서 선택합니다.
2. 작성일자, 품목명, 공급가액을 입력합니다.
3. 세액은 공급가액 입력 시 자동 계산됩니다. 필요시 직접 수정합니다.
4. 발행할 행만 입력하고 나머지는 비워두면 자동으로 건너뜁니다.

---

### 3.3 템플릿 저장 및 불러오기

**저장**
1. 내용을 모두 입력한 후 상단 **저장할 이름** 입력란에 이름을 입력합니다.
2. **저장** 버튼을 클릭합니다.

**불러오기**
1. 상단 드롭다운에서 저장된 템플릿을 선택합니다.
2. **불러오기** 버튼을 클릭합니다.
3. 작성일자만 오늘 날짜로 자동 변경되며 나머지 내용은 그대로 불러옵니다.

**삭제**
- 템플릿 선택 후 **삭제** 버튼을 클릭합니다.

---

### 3.4 일괄 발행 실행 및 결과 확인

1. 입력 완료 후 하단 **일괄 발행** 버튼을 클릭합니다.
2. 각 행의 **발행상태** 열에서 진행 상황을 실시간으로 확인합니다.

| 상태 | 의미 |
|------|------|
| 대기 | 발행 전 |
| 전송중 | 국세청 전송 중 |
| 완료 | 발행 성공 |
| 실패 | 발행 실패 (행에 오류 내용 표시) |

---

이력 목록에서 **승인번호**를 클릭하면 해당 세금계산서를 인쇄할 수 있습니다.

## 6. 사업자 관리

### 6.1 사업자 등록

1. **사업자 관리** 메뉴로 이동합니다.
2. **+ 사업자 추가** 버튼을 클릭합니다.
3. 사업자번호, 상호, 대표자명, 업태, 종목, 이메일을 입력합니다.
4. **저장** 버튼을 클릭합니다.

---

### 6.2 인증서 등록

전자세금계산서 발행을 위해 공인인증서를 등록해야 합니다.

1. 사업자 목록에서 해당 사업자의 **인증서 등록** 버튼을 클릭합니다.
2. 브라우저가 열리며 인증서 등록 페이지로 이동합니다.
3. 화면의 안내에 따라 공인인증서를 등록합니다.

인증서 상태는 **상태 확인** 버튼으로 확인할 수 있습니다.

| 상태 | 의미 |
|------|------|
| 등록됨 | 발행 가능 |
| 미등록 | 인증서 등록 필요 |

---

### 6.3 부가기능 설정

사업자 추가 또는 수정 화면 하단에서 부가기능 사용 여부를 선택할 수 있습니다.
둘 중 하나만 선택 가능합니다.

| 선택 | 설명 |
|------|------|
| 사용 안함 | 부가기능 미사용 (기본값) |
| REC 장기계약 업무 | REC 장기계약 메뉴에서 사용. 계약단가·가중치 입력 필요 |
| 현대건설 VPPA청구 업무 | VPPA청구 메뉴에서 사용. 홈페이지·로그인정보 등 입력 필요 |

**REC 장기계약 업무 설정 항목**

| 항목 | 설명 |
|------|------|
| 계약단가 (원/kWh) | 전력 계약 단가 (소수점 2자리까지 입력 가능) |
| 가중치 | REC 가중치 (예: 1.2) |
| 주거래처 | 세금계산서 발행 시 공급받는자 |

## 7. 비밀번호 변경

1. 화면 우측 상단 타이틀바의 **비밀번호 변경** 버튼을 클릭합니다.
2. 현재 비밀번호를 입력합니다.
3. 새 비밀번호를 입력하고 확인란에 동일하게 입력합니다.
4. **변경** 버튼을 클릭합니다.

## 8. 자주 묻는 질문 (FAQ)

**Q. 발행 실패 시 어떻게 하나요?**
A. 발행 이력 메뉴에서 실패 건을 확인합니다. 오류 메시지를 참고하여 내용을 수정한 후 재발행합니다. 주요 원인은 이메일 형식 오류, 사업자번호 오류, 인증서 미등록입니다.

**Q. 이메일이 거래처에 전달되지 않을 때**
A. 거래처에 등록된 이메일 주소가 올바른지 확인합니다. 스팸 메일함도 확인을 요청하세요.

**Q. 인증서가 만료된 경우**
A. 사업자 관리 메뉴에서 해당 사업자의 **인증서 등록** 버튼을 클릭하여 새 인증서를 등록합니다.

**Q. 같은 거래처에 여러 이메일로 보내고 싶어요**
A. 거래처 관리에서 이메일 항목에 쉼표로 구분하여 여러 주소를 입력합니다.
예: `tax@company.com, manager@company.com`

**Q. 테스트 모드와 운영 모드의 차이는?**
A. 테스트 모드에서는 국세청에 실제 전송되지 않으며 발행 이력만 저장됩니다. 운영 모드에서만 실제 전자세금계산서가 발행됩니다.

**Q. 부가기능 메뉴가 보이지 않아요**
A. 환경 설정 메뉴에서 **부가 기능** 항목의 해당 기능 체크박스를 체크하면 사이드바에 메뉴가 표시됩니다.

## 9. REC 장기계약 (부가기능)

### 9.1 개요

REC(신재생에너지 공급인증서) 장기계약 사업자의 세금계산서를 일괄 발행합니다.
등록된 모든 REC 장기계약 사업자가 한 화면에 행으로 표시되어 한 번에 처리할 수 있습니다.

**사용 전 준비**
1. 환경 설정 → **부가 기능** → "REC 장기계약 업무 사용" 체크
2. 사업자 관리 → 해당 사업자 수정 → **REC 장기계약 업무** 선택 → 계약단가·가중치·주거래처 입력

---

### 9.2 SMP 적용단가 선택

화면 상단에서 전력거래소(KPX) 통합SMP 월별 단가를 조회하고 적용할 월을 선택합니다.

1. 연도를 선택하고 **조회** 버튼을 클릭합니다.
2. 1~12월 단가 표가 표시됩니다.
3. 적용할 월의 **체크박스**를 클릭합니다. (한 달만 선택 가능)
4. 체크된 월의 SMP 단가가 공급가 계산에 자동 적용됩니다.

> 데이터가 없는 월(-로 표시)은 선택할 수 없습니다.

---

### 9.3 발행 그리드

| 열 | 설명 |
|----|------|
| 선택 | 체크박스 — 체크된 행만 발행됩니다 (기본값: 모두 체크) |
| 공급자 | 사업자명 (자동, 변경 불가) |
| 공급받는자 | 주거래처명 (자동, 변경 불가) |
| 종사업장 | 종사업장번호 (자동) |
| 작성일자 | 발행일 (직접 입력) |
| 발행구분 | 청구 / 영수 선택 |
| 품목명 | 직접 입력 (기본값: 발전소명 REC 대금) |
| REC발급 | REC 발급 수량 입력 |
| 공급가(원) | 자동 계산 (수정 불가) |
| 세액(원) | 자동 계산 (수정 불가) |
| 발행상태 | 대기 / 전송중 / 완료 / 실패 |

**공급가 계산 공식**
```
공급가 = (계약단가 - 통합SMP) × 1000 / 가중치 × REC발급수량
세액   = 공급가 × 10%
```
- 공급가가 음수인 경우 0으로 처리됩니다.
- 거래처별로 소수점 처리 방식이 다를 수 있습니다 (버림 또는 반올림, 시스템 설정에서 관리).

---

### 9.4 세금계산서 발행

1. SMP 월 체크박스 선택 후 각 행의 REC발급 수량을 입력합니다.
2. 공급가와 세액이 자동 계산됩니다.
3. 발행하지 않을 행은 좌측 체크박스를 해제합니다.
4. **발행하기 →** 버튼을 클릭합니다.
5. 해당 월에 동일 공급자·공급받는자의 기발행 내역이 있으면 경고창이 표시됩니다.
   - **확인**: 계속 발행
   - **취소**: 해당 행 건너뜀

## 10. 현대건설 VPPA청구 (부가기능)

### 10.1 개요

신재생 에너지(태양광 등) 발전소의 REC(신재생에너지 공급인증서) 판매 청구 업무를 관리합니다.
사업자 관리에서 **현대건설 VPPA청구 업무** 옵션을 선택한 사업자에 한해 사용할 수 있습니다.

**업무 흐름**
```
청구 등록 → 제출 → 승인 확인 → 세금계산서 발행
```

**사용 전 준비**
1. 환경 설정 → **부가 기능** → "현대건설 VPPA청구 업무 사용" 체크
2. 사업자 관리 → 해당 사업자 수정 → **현대건설 VPPA청구 업무** 선택 → 관련 정보 입력

---

### 10.2 사업자 설정

사업자 관리에서 해당 사업자의 VPPA 관련 항목을 등록합니다.

| 항목 | 설명 |
|------|------|
| 홈페이지 주소 | 청구 사이트 URL |
| 아이디 / 비밀번호 | 청구 사이트 로그인 정보 |
| 계약단가 (원/kWh) | 전력 계약 단가 |
| 청구서 생성 URL | 청구서 작성 폼 직접 이동 주소 |
| 가중치 | REC 가중치 (예: 1.2) |
| 주거래처 | 세금계산서 발행 시 공급받는자 |

---

### 10.3 청구 등록

| 항목 | 설명 |
|------|------|
| 사업자 | 발전소(사업자) 선택 |
| 청구서 발행월 | 청구서를 발행하는 월 |
| 생산월 | 실제 전력 생산 월 |
| 발전량(kWh) | 해당 월 발전량 |
| REC수량 | 공급인증서 수량 |
| 공급가액 | 청구 금액 |
| 공급인증서 | PDF 파일 첨부 |
| SMP 고지서 | 전력거래실적 고지서 텍스트 붙여넣기 후 PDF 생성 |

---

### 10.4 청구서 제출 (자동)

**제출** 버튼 클릭 시 청구 사이트에 자동으로 접속하여 양식을 작성합니다.

1. 청구 사이트 자동 로그인
2. 발행월 / 생산월 자동 선택
3. 발전량, 공급가액, REC수량, 가중치 자동 입력
4. SMP 파일 및 공급인증서 자동 첨부
5. **최종 확인창 표시** → 내용 확인 후 팝업 창에서 **임시저장** 버튼 직접 클릭

---

### 10.5 청구 상태

| 상태 | 의미 |
|------|------|
| 미제출 | 청구서를 아직 제출하지 않은 상태 |
| 제출완료 | 청구 사이트에 정상 제출됨 |
| 승인완료 | 현대건설에서 승인 및 정산금액 확정 |
| 오류 | 제출 실패 (수정 후 재시도) |
| 발행완료 | 세금계산서 발행까지 완료 |

- **상태 텍스트 클릭** 시 해당 사업자의 청구 사이트가 자동 로그인 후 열립니다.
- **상태 확인 갱신** 버튼으로 승인 상태와 정산금액을 자동으로 가져옵니다.

---

### 10.6 세금계산서 발행

승인완료 상태에서 **세금계산서 발행** 버튼을 클릭합니다.

- **공급자**: 해당 발전소(사업자)
- **공급받는자**: 사업자에 설정된 주거래처
- **품목명**: `발전소명 N월 REC 대금` (예: `한국 2월 REC 대금`)
- **금액**: 승인된 공급가액 및 세액 적용

> 발행완료 건의 수정이 필요한 경우 **발행이력** 메뉴에서 수정발행 또는 취소 처리하세요.
